Afghanistan from Deadline, but larger.
Features as of 0.2-pre:
- Normal spectator drones
- Trees
- Buildings
- Grassy valleys and sandy hills - procedural perlin noise terrain generation, thanks to Egomoose, triangles are made possible. Perlin noise hill meshing is sourced from some tutorials and online resources.
- 50% less recoil to compensate for large map (machine guns are slightly more possible now)
- 8 magazines because you're going on a long mission
- Wave spawning system with 100 players before no respawning is allowed. Inspired from Insurgency Sandstorm's mechanics

This repo is expected to run in ROBLOX > DEADLINE by Recoil Group - Private Server Luau Scripting Functionality
