Afghanistan from Deadline, but larger.
Features as of 0.2-pre:
- Spectator drones that play music instead of drone noises
- Trees
- Buildings
- Grassy valleys and sandy hills - procedural perlin noise terrain generation, thanks to Egomoose, triangles are made possible. Perlin noise hill meshing is sourced from some tutorials
- 50% less recoil to compensate for large map (machine guns are slightly more possible now)
- 8 magazines
- Wave spawning system with 100 players before no respawning

This repo is expected to run in ROBLOX > DEADLINE by Recoil Group - Private Server Luau Scripting Functionality
